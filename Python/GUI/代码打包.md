#### 代码打包以及分发
  
1.注意资源路径，代码中的当前目录路径用下面函数获取
```
import sys
import os

def resource_path(relative_path):
    """获取资源文件路径, 兼容打包和开发环境"""
    if hasattr(sys, '_MEIPASS'):  # PyInstaller 解包目录
        base_path = sys._MEIPASS
    else:
        base_path = os.path.abspath(".")
    return os.path.join(base_path, relative_path)

```

2.打包
```
假设目录：
APP
   --build
   --dist
   --resources
   --configs
   main.py
```

```
pip install pyinstaller
```

```
pyinstaller --noconsole  --add-data "resources;resources" --add-data "configs;configs" --name PddSQL main.py
# 最终会打包在dist目录里
```

3.分发到windows安装
```
安装Inno Setup，并打开，编辑配置文件并运行，windows安装包将会在build-> Output Folder
```
```
[Setup]
AppName=PddSQL
AppVersion=1.0
DefaultDirName={pf}\PddSQL
DefaultGroupName=PddSQL
OutputBaseFilename=PddSQL_Installer
Compression=lzma
SolidCompression=yes
AllowNoIcons=yes

[Files]
Source: "具体路径\dist\PddSQL\*"; DestDir: "{app}"; Flags: ignoreversion recursesubdirs createallsubdirs

[Icons]
Name: "{group}\PddSQL"; Filename: "{app}\PddSQL.exe"
Name: "{userdesktop}\PddSQL"; Filename: "{app}\PddSQL.exe"  ;
```
